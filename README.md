# Superstore Sales Analysis Dashboard

## 📊 Project Overview
This project is a Power BI dashboard created using the Superstore dataset to analyze sales performance.

## 🔍 Key Insights
- West region has highest sales
- Technology category gives more profit
- Sales trend increased over years

- Power BI
- Data Cleaning
- Data Visualization
